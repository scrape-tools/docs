<CodeGroup>
  ```bash curl
  curl -H "Authorization: Bearer <key>" "https://api.scrape.rocks/musixmatch/lyrics?title=Bohemian%20Rhapsody&artist=Queen"
  ```
  ```python Python
  import requests

  headers = {"Authorization": "Bearer <key>"}
  r = requests.get("https://api.scrape.rocks/musixmatch/lyrics?title=Bohemian%20Rhapsody&artist=Queen", headers=headers)
  print(r.json())
  ```
  ```javascript JavaScript
  fetch("https://api.scrape.rocks/musixmatch/lyrics?title=Bohemian%20Rhapsody&artist=Queen", {
    headers: { Authorization: "Bearer <key>" }
  })
    .then(r => r.json())
    .then(console.log)
  ```
  ```typescript TypeScript
  const response = await fetch("https://api.scrape.rocks/musixmatch/lyrics?title=Bohemian%20Rhapsody&artist=Queen", {
    headers: { Authorization: "Bearer <key>" }
  });
  const data = await response.json();
  console.log(data);
  ```
