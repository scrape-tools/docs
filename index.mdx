## Base URL

```
https://bridge.scrape.rocks/
```

## Authentication

All requests need a Bearer token in the `Authorization` header:

```
Authorization: Bearer <your-api-key>
```

<Callout type="warning">
  Don't have a key? Message **@mwpk** on Discord.
</Callout>
